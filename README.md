
# Projeto Web – Design de Produto HTML

## 📌 Português

### Visão Geral

Este repositório contém um projeto em **HTML** desenvolvido com foco em **performance, fluidez e design atemporal**, inspirado em experiências modernas de navegação, como o site **apple.com**.

O projeto foi pensado para rodar de forma **leve e eficiente na web**, além de servir como base para uma **futura adaptação em aplicativo mobile**, mantendo consistência visual e experiência do usuário entre plataformas.

O desenvolvimento foi feito em etapas progressivas, com apoio conceitual da inteligência artificial **Claude (https://claude.ai/)**.

---

### 🧠 Etapas do Desenvolvimento

#### ✅ Etapa 1 — Estrutura Inicial (`index.html`)

Primeira versão conceitual do projeto, com foco em:

- Design de produto
- Apresentação de projetos
- Conceito de uso de **peças e modelos 3D**
- Layout fluído, moderno e atemporal
- Estrutura base em HTML

O arquivo **`index.html`** representa essa fundação inicial.

---

#### ✅ Etapa 2 — Otimização de Performance (`index2.html`)

Na segunda etapa, o foco foi **melhoria de performance e usabilidade**, com as seguintes alterações:

- **Redução da quantidade de elementos gráficos**
- **Correção do problema de rolagem infinita**
- Simplificação do layout e da estrutura
- Melhor controle do fluxo e navegação da página

O resultado foi o arquivo **`index2.html`**, uma versão mais leve e otimizada do projeto.

---

#### ✅ Etapa 3 — Refinamento Visual e Conceitual (`index3.html`)

Na terceira etapa, o projeto passou por ajustes visuais e conceituais, com foco em identidade e temática do produto. Foram solicitadas as seguintes mudanças:

- **Alteração das cores de fundo** para uma paleta próxima ao **cinza chumbo**
- Em **"Projetos Recentes"**, substituição do conteúdo por **peças de performance automotiva produzidas em impressora 3D**
- **Substituição da cor dourada por verde lima**, reforçando um visual mais técnico e moderno

Essas modificações resultaram no arquivo **`index3.html`**, que representa uma versão mais madura do projeto, com identidade visual mais forte e alinhada ao conceito de engenharia, performance e design industrial.

---

## 🌎 English

### Overview

This repository contains an **HTML project** focused on **performance, smooth navigation, and timeless design**, inspired by modern web experiences such as **apple.com**.

The project is designed to run **efficiently on the web**, while also serving as a foundation for a **future mobile application**, maintaining consistent visual identity and user experience.

The development followed progressive stages, with conceptual support from **Claude AI (https://claude.ai/)**.

---

### 🧠 Development Stages

#### ✅ Stage 1 — Initial Structure (`index.html`)

The first conceptual version focused on:

- Product design concepts
- Project presentation
- Use of **3D models and components**
- Fluid, modern, and timeless layout
- Base HTML structure

---

#### ✅ Stage 2 — Performance Optimization (`index2.html`)

The second stage focused on **performance and usability**, including:

- **Reduced number of graphic elements**
- **Infinite scrolling issue fixed**
- Layout and structure simplification
- Improved navigation flow

This resulted in the **`index2.html`** file.

---

#### ✅ Stage 3 — Visual & Conceptual Refinement (`index3.html`)

The third stage focused on refining the visual identity and project theme, with the following changes:

- **Background colors updated** to a **dark charcoal gray palette**
- In **"Recent Projects"**, content replaced with **automotive performance parts produced via 3D printing**
- **Gold accents replaced with lime green**, reinforcing a technical and modern aesthetic

These changes resulted in the **`index3.html`** file, representing a more mature and conceptually aligned version of the project.
